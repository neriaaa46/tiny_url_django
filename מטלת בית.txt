כתובת URL מקוצרת יכולה למספר כתובות URL מלאות , שים לב איך אתה יוצר את הכתובת מה יכול להשתבש איך ניתן לטפל 

תתחזק מונה לכל כתובת מקוצרת שים לב איך כדי להגדיל את המונה 

- בדיקה ליצירת והפניה מחדש של כתובת URL מקוצרת 
- בדיקה לאיתור כתובת URL קצרה שאינה קיימת 
- אם נדרש עוד בדיקות בצע אותן . 

כתיבה יפה והיערות במקום שצריך . 

השתמש ב API הנתון בכדי ליצור URL מקוצר 

pip freeze > requirements.txt



python tiny_url\manage.py makemigrations

python tiny_url\manage.py migrate

python tiny_url\manage.py runserver
